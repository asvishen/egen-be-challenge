# egen-be-challenge

This is a Personal Weight Tracker Application built using Spring Boot API. 
## Model
- Metric(weight)
- Alert(Over or Under weight)

## Features

- Adds a new Metric
- Read all Metrics
- Reads metrics between timestamps
- Read all Alerts
- Read alerts between timestamps

## Dependencies

- Spring Boot API
- JUnit
- Mockito
- EasyRules
- Morphia
- Maven

## Execution

```java
mvn spring-boot:run 
```
## Java Version

- 1.8
